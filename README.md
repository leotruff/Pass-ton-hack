# Passe Ton Hack - Mini-site d'equipe

Equipe : **Les Chevaliers des Bits**

## Structure du projet

- `index.html` : page d'accueil equipe
- `style.css` : feuille de style unique et commune
- `leo-truffin.html`
- `alex-alsberghe.html`
- `louis-hostelet.html`
- `antoine-leurent.html`
- `assets/eleves/prenom-nom/` : captures d'ecran par eleve

## Rappels de conformite

- HTML semantique utilise : `header`, `main`, `section`, `footer`
- Un seul `h1` par page
- Navigation retour depuis chaque page eleve vers `index.html`
- Captures integrees dans `figure` + `figcaption`
- Aucune donnee sensible dans les contenus

## Convention captures

Exemple de nommage attendu :
- `p1_01.png`, `p1_02.png`, `p1_03.png`
- `p2_01.png`, `p2_02.png`, `p2_03.png`
